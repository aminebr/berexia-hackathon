# ReactApp

This application allows the user to execute some operation on csv/xsls files without using code, just dragging
and dropping to build the request.

## Front-end

Used technologies:

- Reactjs
- DiagramEngine

## Back-end

Used technologies:

- Play framework
- Spark
- Scala
